# C Programming Practice

This repository contains a collection of C programs created while learning fundamental programming concepts and data structures.

The programs cover topics such as searching algorithms, sorting algorithms, linked lists, stacks, queues, trees, graphs, recursion, strings, and matrix operations.

## Topics Covered

### Searching Algorithms
Programs implementing searching techniques.

Examples:
- linearsearch.c
- linearsearch_function.c
- linearsearch_linkedlist.c
- binarysearch1.c
- binarysearch2.c
- target_search.c

### Sorting Algorithms
